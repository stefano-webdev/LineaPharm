# Progetto WordPress B2B Pharma

## Descrizione
Questo progetto è un sito vetrina B2B nel settore farmaceutico, realizzato con **WordPress**, **Elementor** e codice front-end personalizzato.  
L'obiettivo è stato creare un sito professionale, accessibile e responsive da usare come **progetto portfolio** per dimostrare competenze in sviluppo WordPress, Elementor e front-end in generale.

Il progetto include:  
- Homepage strutturata con hero video, presentazione azienda, servizi, prodotti, qualità e certificazioni, numeri chiave e call-to-action.  
- Form di contatto con validazione, checkbox privacy policy e gestione keyboard-friendly (tab, enter, space).  
- Layout **responsive desktop-first** con Elementor.  
- Stile visivo corporate pharma: colori blu primari e secondari, molto spazio bianco, tono sobrio e professionale.

## Tecnologie utilizzate
- WordPress con tema custom child 
- Elementor per le pagine 'Chi siamo', 'Servizi', 'Qualità', 'Contatti' e 'Privacy'
- Gutenberg per la Homepage e i template globali come header e footer
- PHP base per le funzionalità necessarie  
- HTML5 / CSS3 / JS per interattività front-end   
- Git / GitHub per il versionamento  

## Accessibilità e UX
- Tutti gli input hanno label e validazione  
- Checkbox privacy policy gestito con keyboard-friendly (space per attivare/disattivare)  
- Bottone di invio nel form di contatto abilitato solo se tutti i campi sono validi  
- Tab order coerente e funzionale su tutte le pagine
- Bottoni e testi coerenti, chiari e con un buon contrasto colori
